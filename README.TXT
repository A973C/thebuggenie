* INSTALLATION *

via web
-------
To install BUGS 2, copy this folder to your webserver, and visit the location in your web-browser.
BUGS 2 should automatically forward you to the installation script, but if it doesn't, you can visit
install.php manually.


via command-line (unix/linux only)
----------------------------------
You can use the B2 command-line client to install, if you prefer that. Run "./b2" from this folder
To install, run "./b2 install"
