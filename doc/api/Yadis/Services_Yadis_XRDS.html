Class Services_Yadis_XRDS

Description
Description | Methods (details)

This class performs parsing of XRDS documents.

You should not instantiate this class directly; rather, call parseXRDS statically:

  $xrds = Services_Yadis_XRDS::parseXRDS($xml_string);

If the XRDS can be parsed and is valid, an instance of Services_Yadis_XRDS will be returned. Otherwise, null will be returned. This class is used by the Services_Yadis_Yadis::discover method.

		services (line 164)
	
 
	
	

Returns a list of service objects which correspond to <Service>  elements in the XRDS XML document for this object.


Optionally, an array of filter callbacks may be given to limit  the list of returned service objects.  Furthermore, the default  mode is to return all service objects which match ANY of the  specified filters, but $filter_mode may be  openidActions::SERVICES_YADIS_MATCH_ALL if you want to be sure that the  returned services match all the given filters.  See Services_Yadis_Yadis for detailed usage information on filter  functions.

	
    
				
  • return: An array of Services_Yadis_Service objects if $filter_mode is a valid  mode; null if $filter_mode is an invalid mode (i.e., not  openidActions::SERVICES_YADIS_MATCH_ANY or openidActions::SERVICES_YADIS_MATCH_ALL).
