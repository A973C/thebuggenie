  1. <b>Benutzer zu Editionen oder Komponenten zuweisen</b><br>
  2. Auf der Übersichts-Seite der Editionen, Builds & Komponenten können Sie nach Entwicklern/Benutzern suchen.
  3. Geben Sie die Details, nach denen Sie suchen, ein und klicken Sie anschließend auf den Namen des Benutzers wenn dieser erscheint.
  4. Dadurch erschint eine Auswahlliste, bei der Sie auswählen können, wo Sie den Benutzer zuweisen möchten.
  5. Sobald ein Sachverhalt für den Teil des Produktes, zu dem der Benutzer zugewiesen wurde, wird dieser Benutzer benachrichtigt.
  6. Dasselbe passiert, wenn ein Sachverhalt aktualisiert wird.
  7. Benutzer in dieser Liste erscheinen ebenfalls unter "Produkt Entwickler" auf der Übersichts-Seite des Produktes.
