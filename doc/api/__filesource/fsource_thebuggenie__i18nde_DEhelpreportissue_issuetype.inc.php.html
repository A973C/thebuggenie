  1. <div style="padding: 2px;"><b>Was ist eine "Art des Sachverhaltes"?</b><br>
  2. Eine "Art des Sachverhaltes" beschreibt den Sachverhalt, den Sie melden. Durch die richtige Auswahl der 
  3. "Art des Sachverhaltes" können Sie sicher gehen, dass der richtige Entwickler sofort benachrichtigt wird.
  4. <br>
  5. <br>
  6. Es können unterschiedliche Arten zur Auswahl stehen, aber diese sind normalerweise ähnlich zu "Fehlermeldung", 
  7. "Verbesserungsvorschlag", usw. 
  8. Wählen Sie die Art, die am besten zu Ihrem Sachverhalt, den Sie gerade melden, passt.<br>
  9. <br>
  10. <br>
  11. <b>Bsp: </b>Sie haben einen Fehler in der Software, die Sie gerade benutzen, gefunden,<br>
  12. und Sie wollen es den Entwicklern melden, damit diese den Fehler beheben können.<br>
  13. <i><u>Wählen Sie "Fehlermeldung"</u></i><br>
  14. <br>
  15. <b>Bsp 2:</b>Sie vermissen eine bedeutende Funktion in der Software.<br>
  16. Sie wollen es den Entwicklern melden, damit diese die Funktion in einer zukünftigen Version hinzufügen können.<br>
  17. <i><u>Wählen Sie "Verbesserungsvorschlag"</u></i><br>
