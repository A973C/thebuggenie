  1. <div style="padding: 2px;"><b>Was ist eine Komponente?</b><br>
  2. Eine Software wird normalerweise in unterschiedliche "Teile", sogenannte Komponenten, aufgeteilt. Durch das 
  3. Aufteilen der Software in diese Komponenten, können Entwickler oder ein Team von Entwicklern sich auf einen bestimmten 
  4. Teil der Software konzentrieren, während andere Entwickler sich auf andere Teile konzentrieren können.
  5. <br>
  6. <br>
  7. Wenn Sie die richtige Komponente auswählen, erhöht sich die Wahrscheinlichkeit, dass der richtige Entwickler sofort 
  8. benachrichtigt wird. Somit erhöht sich auch die Chance, dass der Sachverhalt richtig bearbeitet wird.<br> 
  9. <br>
  10. <b>Wie weiß ich, welche Komponente ich wähle?</b><br>
  11. Die Komponenten haben normalerweise selbsterklärende Namen, wie "Menü", "Drucken", usw - dies hängt natürlich davon ab, 
  12. welche Art von Software Sie benutzen und kann somit alles sein. Schauen Sie nach einer Komponente, die Ihrer Meinung 
  13. nach die passende für Ihren Sachverhalt ist.
  14. <i>Wenn Sie keine passende Komponente finden können</i>, wählen Sie die Komponente "Allgemein" oder "Unbekannt" 
  15. - diese sind meistens vorhanden.<br>
