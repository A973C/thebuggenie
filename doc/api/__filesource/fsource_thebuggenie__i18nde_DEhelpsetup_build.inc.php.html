  1. <b>Eine neue Build erstellen</b><br>
  2. Builds werden auf der Einstellungs-Seite der Editionen hinzugefügt. 
  3. Unterhalb der Details zur Edition finden Sie "Neue Build hinzufügen".
  4. Hier geben Sie den Namen und die Versionsnummer(n) der neuen Build ein.
  5. Jede Versionnummer die leer gelassen wird, erhält automatisch die 0.
  6. Klicken Sie auf "Hinzufügen" und die neue Build erscheint automatisch in der Liste.
  7. <br>
  8. <br>
  9. Bei den Builds können Sie verschiedene Aktionen ausführen, was durch die verschiedenen Symbole unter dem Namen der Build verdeutlicht wird.
  10. Durch Überfahren der Symbole mit dem Mauszeiger sehen Sie eine Beschreibung zu diesen Symbolen.<br>
  11. <br> 
  12. <b>WICHTIG: Denken Sie daran anderen Benutzern Zugriff auf dieses Produkt zu geben.</b><br>
  13. Der Zugriff wird der Gruppe, in der Sie sich befinden, erteilt.
  14. Aber denken Sie daran, dass Sie anderen Gruppen ebenfalls Zugriff garantieren.
