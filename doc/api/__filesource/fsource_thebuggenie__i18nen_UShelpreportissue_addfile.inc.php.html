  1. <div style="padding: 2px;"><b>How do I add a file to my report?</b><br>
  2. If you want to include a file with your issue report, click the "Browse"-button next to the <b>File</b>-field. A file browser will pop up. Navigate to
  3. the file you want to include, and select it by double-clicking on it, or single-click it and then press ENTER. The dialog will then disappear, and the
  4. filename will be displayed in the <b>File</b>-field.<br>
  5. <br>
  6. Remember to also fill in a short description about the file you are attaching.<br>
  7. <br>

  9. There are some rules that you should be aware of when uploading files:
  10. <ul>
  11.     <li>Files may need to be below a certain size</li>
  12.     <li>Files may need to be of a certain type</li>
  13. </ul>
  14. Look for the text that says anything about these restrictions, below the <b>Description</b>-field, above the "Attach"-button.<br>
