  1. <div style="padding: 2px;"><b>Was sollte eine Zusammenfassung enthalten?</b><br>
  2. Eine Zusammenfassung sollte den Sachverhalt in einem kurzen Satz oder einigen Worten beschreiben.
  3. Gute Zusammenfassungen vereinfachen die Identifizierung des Problems oder des Sachverhaltes, den Sie melden 
  4. und helfen den Entwicklern sich auf die Lösung des Problems zu konzentrieren, anstatt herauszufinden, was Sie meinen.
  5. <br>
  6. <br>
  7. <b>Einige Beispiele für <u>gute</u> Zusammenfassungen?</b>
  8. <ul>
  9.     <li>Programm X hängt sich auf wenn versucht wird PDF-Dateien zu öffnen</li>
  10.     <li>Fehlermeldung erscheint nach Drücken des "Drucken"-Buttons</li>
  11.     <li>Rechtschreibfehler im Titel auf Seite Y</li>
  12. </ul>
  13. <br>
  14. <b>Einige Beipiele für <u>schlechte</u> Zusammenfassungen?</b>
  15. <ul>
  16.     <li>Programm X hängt sich auf</li>
  17.     <li>Fehlermeldung erscheint die ganze Zeit!!!!!!!</li>
  18.     <li>Das Programm ist buggy</li>
  19. </ul>
