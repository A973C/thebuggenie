  1. <div style="padding: 2px;"><b>Wie füge ich eine Datei zu meinem Sachverhalt hinzu?</b><br>
  2. Wenn Sie eine Datei zu Ihrem Sachverhalt hinzufügen wollen, klicken Sie auf den "Durchsuchen" Button neben dem <b>Datei</b>-Feld.
  3. Ein Datei-Auswahlfenster erscheint. Navigieren Sie zu der Datei, die Sie anhängen wollen und wählen Sie diese duch 
  4. Doppelklicken oder durch einfaches Klicken und das Betätigen des OK Buttons, aus. Das Fenster verschwindet anschließend
  5. und der Dateiname der gewählten Datei wird in dem <b>Datei</b>-Feld angezeigt.<br>
  6. <br>
  7. Denken Sie daran ebenfalls eine kurze Beschreibung über die angehängte Datei anzugeben.<br>
  8. <br>
  9. <b>Folgendes sollte beim hochladen von Dateien beachtet werden</b><br>
  10. Es existieren einige Regeln, die Sie beachten sollten, wenn Sie Dateien hochladen:
  11. <ul>
  12.     <li>Dateien müssen vielleicht unter einer bestimmten Größe sein</li>
  13.     <li>Dateien müssen vielleicht ein bestimmtes Format besitzen</li>
  14. </ul>
  15. Beachten Sie den Text unter dem <b>Beschreibungs</b>-Feld und über dem "Anhängen"-Button, der Sie über eventuelle 
  16. Beschränkungen informiert.<br>
