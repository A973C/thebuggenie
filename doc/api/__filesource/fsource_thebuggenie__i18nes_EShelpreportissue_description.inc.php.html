  1. <div style="padding: 2px;"><b>¿Qué debe contener una descripción?</b><br>
  2. La descripción debe dar una buena y completa explicación del problema que está informando. Puntos que pueden estar en la descripción:
  3. <ul>
  4.     <li>porque está informando el problema</li>
  5.     <li>qué sucedió - cual es el problema</li>
  6.     <li>cual sería el comportamiento esperado si el problema no existiera</li>
  7. </ul>
  8. Recuerde que por lo general uno no es demasiado detallado - las descripciones más completas y detalladas son mejores.<br>
  9. También recuerde que puede utilizar la barra de herramientas para dar formato al texto y lo ayudará para describir los pasos para reproducir el problema.<br>
