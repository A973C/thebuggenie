  2. Bevor Sie Sachverhalte melden können, benötigen Sie mindestens eine Edition in Ihrem Produkt. Wenn Sie mehrere

  19. Um eine neue Edition zu erstellen, geben Sie in dem Feld "Eine Edition hinzufügen" den Namen der Edition ein
  20. und drücken Sie den Plus-Button auf der rechten Seite um die Edition hinzuzufügen. Die Edition wird in der Liste
  21. erscheinen - klicken Sie auf den Namen. Auf der neuen Seite, auf die Sie gelangen, können Sie Informationen der 
  22. Edition bearbeiten, welche als Information für das Produkt angezeigt werden. Die Edition hat ebenfalls Einstellungen, 
  23. welche die selben sind, wie die vom Produkt. (Lesen Sie mehr über diese Einstellungen unter "Einrichten eines Produktes")

  33. Es gibt zwei Möglichkeiten um Komponenten zu erstellen. Sie können diese direkt von der Übersichts-Seite der 
  34. Editionen und Komponenten, die Sie am Anfang dieses Themas finden, erstellen oder auf dr Einstellungs-Seite der 
  35. Edition. Durch das Hinzufügen einer Komponente auf der Einstellungs-Seite der Edition, wird diese Komponente 
  36. automatisch zur edition hinzugefügt. Wenn Sie diese auf der Übersichts-Seite der Editionen und Komponenten 
  37. hinzufügen, müssen Sie die Komponente noch zur Edition hinzufügen, bevor Sie für Benutzer verfügbar ist.
  38. Sie können diese auf der Einstellungs-Seite der Edition, durch Klicken auf "Hinzufügen" neben dem
  39. Namen der Komponente hinzufügen. (Sie gelangen durch Klicken auf den Namen der Edition innerhalb der Übersichts-Seite
  40. der Editionen und Komponenten auf die Einstellungs-Seite der Edition.)
